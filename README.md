# GameHelper3366
this is a game help tool, help user to play hige score on 3366.com

game url is [游戏地址] <a>http://www.3366.com/flash/1000132.shtml</a>

    本软件的实现的功能是针对网页上的这个flash游戏，实现自动玩游戏的功能，主要步骤是截取桌面游戏信息，然后分析出玩法，并同步显示步序。
